This section (and the following sections) is / are part of the "old Angular course".

The following sections were recorded with older Angular versions (i.e., <18). They do not use standalone components and signals, for example.

Therefore, you can ignore these old sections (and mark them as "Completed" manually, if course progress is important for you).

But if you know that you'll be working with an older Angular version, the following sections may be very valuable for you.

Angular has a quite fragmented version landscape - there are many Angular projects that still use Angular 9, 10 etc. That's why I decided to keep this old course online for now. Eventually, once I see consumption drop significantly, these old sections will be removed (I will, of course, send an announcement before doing that, though). But for the time being, they stay online to teach you how to work with older Angular versions.

For this and the next sections, it's worth noting that you should follow along with the attached starting project since a brand-new Angular project would be missing some setup steps that are required for this and the next sections.
